## بخش پنجم: آزمایشگاه محاسباتی در جیب؛ راه‌اندازی لینوکس بومی و ابر محاسباتی روی موبایل

### ۱. کشف و چالش: شبیه‌سازی بدون نیاز به کامپیوتر
در آموزش‌های گذشته، کدهای مربوط به فرضیه‌های کافمن و ساترلند در محیط لینوکس روی دسکتاپ اجرا شدند. اما پردازنده‌های موبایل‌های امروزی توان پردازشی فوق‌العاده‌ای دارند و می‌توانند به عنوان آزمایشگاه محاسباتی مستقل عمل کنند. برخی از دانشجویان در مراحل قبلی هنگام نصب پایتون و کتابخانه‌های علمی روی ترموکس با خطاهایی مواجه می‌شدند. این خطاها عمدتاً ناشی از اختلال در مخازن پیش‌فرض ترموکس در ایران یا فعال نبودن پوسته مخازن جانبی (Universe) در نسخه مینیمال اوبونتو است.

در این بخش، با برطرف کردن این قیدهای فنی، بستر محاسباتی بدون خطا روی موبایل مستقر می‌شود تا امکان اجرای کدهای منشاء حیات در هر مکانی فراهم آید.

---

### ۲. طبقه‌بندی و ساختار: مراحل راه‌اندازی لینوکس بومی با ترموکس (Termux)
برای راه‌اندازی بدون خطای آزمایشگاه جیبی، مراحل زیر را به ترتیب روی گوشی اندرویدی خود اجرا کنید. تمامی باکس‌های دستوری در این متن قابلیت کپی سریع دارند.

#### گام اول: حل مشکل مخازن ترموکس
ابتدا ترموکس را از اف-دروید (F-Droid) یا مخزن رسمی گیت‌هاب آن دانلود کنید (نسخه گوگل‌پلی قدیمی است و خطا می‌دهد). پس از باز کردن برنامه، دستور زیر را برای تغییر مخزن به سروری پایدار وارد کنید، گزینه‌ی **Main Repository** را انتخاب کرده و میرور را روی منبعی پایدار (مانند Aliyun یا BFSU) قرار دهید:

```bash
termux-change-repo
```

پس از تغییر میرور، با دستور زیر مخازن ترموکس را به‌روزرسانی کنید:

```bash
pkg update && pkg upgrade -y
```

#### گام دوم: نصب و ورود به اوبونتو
با استفاده از ابزار `proot-distro` نسخه کامل لینوکس اوبونتو را بدون نیاز به روت کردن گوشی مستقر کنید:

```bash
pkg install proot-distro -y
proot-distro install ubuntu
proot-distro login ubuntu
```

#### گام سوم: فعال‌سازی مخازن علمی اوبونتو و نصب پایتون
پس از ورود به اوبونتو، برای دسترسی به پکیج‌های علمی پایتون (SciPy و Matplotlib)، باید پوسته **Universe** را فعال کنید. دستورات زیر را به ترتیب اجرا کنید:

```bash
apt update && apt upgrade -y
apt install software-properties-common -y
add-apt-repository universe
apt update
```

اکنون پایتون ۳ و تمام کتابخانه‌های محاسباتی مورد نیاز را بدون هیچ خطایی نصب کنید:

```bash
apt install python3 python3-pip python3-numpy python3-scipy python3-matplotlib nano -y
```

### ۳. مدل‌سازی و داده‌های تجربی: اسکریپت تست بستر
پس از نصب موفقیت‌آمیز، باید عملکرد کتابخانه‌ها را ارزیابی کنیم. درون ترمینال اوبونتو با دستور `nano test_native.py` فایلی بسازید، کدهای پایتونی را که در باکس زیر قرار دارند درون آن کپی و ذخیره کنید:

```python
# Verification script for mobile environment testing
import numpy as np
import scipy
import matplotlib

print("Checking native environment inside Android Ubuntu...")
print(f"NumPy Version: {np.__version__}")
print(f"SciPy Version: {scipy.__version__}")
print("Native environment setup is 100% successful!")
```

پس از ذخیره، با دستور زیر آن را روی گوشی اجرا کنید:

```bash
python3 test_native.py
```

در صورت نصب صحیح، نسخه‌های کتابخانه‌ها و پیام موفقیت‌آمیز بودن تنظیمات روی صفحه ترمینال موبایل شما چاپ خواهد شد.
